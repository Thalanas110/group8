
  # ExamHub Frontend

  Frontend application for the online examination system.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Vercel

  Deploy this folder as the Vercel project root. The project includes `vercel.json` with the Vite build command, `dist/client` output directory, and SPA route rewrite.

  Set `VITE_PHP_BASE_URL` in Vercel to the Render backend API URL.
